About US

We are a team from SJTU-JI. This website is our capstone project which is aimed at to use NLP and aspect level sentiment analysis to design an Evaluation system for the Rooms in Airbnb.

The users can give the comments to those rooms and we can return the analysis results. And we can show a radar chart for each room to make your choice easier.
